> Utility modules shared between [Dext](https://github.com/vutran/dext) and [Dext Package Manager](https://github.com/vutran/dext-cli).

## Install

With `npm`:

```bash
$ npm install --save dext-core-utils
```

With `yarn`:

```bash
$ yarn add dext-core-utils
```
